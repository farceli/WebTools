# 工具 - 智能日历聚合平台

一个以智能日历为核心的工具聚合平台，集成了农历、节假日、实时时钟等丰富功能，并提供多种实用工具。

## 🗓️ 核心功能 - 智能日历

### 日历功能特性
- 📅 **完整日历视图** - 支持月视图和年视图切换
- 🌙 **农历信息** - 显示农历日期、传统节日、二十四节气
- 🎌 **节假日标注** - 基于官方数据的假期和调休标识
- ⏰ **实时时钟** - 动态显示当前时间、日期、周几和农历
- 🏷️ **智能标识** - "今"、"休"、"班" 日期角标
- 📱 **响应式设计** - 完美适配桌面、平板、手机
- 🎨 **简约美观** - 现代化界面设计，用户体验优先

### 日历数据特性
- 📊 **官方数据源** - 基于 [holiday-cn](https://github.com/NateScarlet/holiday-cn) 项目
- 🕒 **时区准确** - 修复时区问题，确保日期准确性
- ⚡ **本地存储** - 所有数据本地化，无需网络请求

## 🛠️ 附加工具

- 🖼️ **图片生成器** - 生成指定大小的随机图片，支持 PNG、JPEG、WEBP、BMP 格式
- 🎬 **视频生成器** - 生成指定大小的随机视频文件，支持 MP4、WEBM 格式
- 📎 **附件生成器** - 生成 PDF、XLSX、DOCX 等格式的测试文件
- 📊 **数据生成器** - 生成各种测试数据（中文姓名、身份证号、手机号、邮箱、公司名称、社会统一信用代码等）
- 📝 **文字统计器** - 统计文本的字数、词数、行数等信息，支持多种统计模式

## 📅 假期数据管理

### 新增假期数据文件

如果需要添加新年份的假期数据，请按以下步骤操作：

#### 1. 下载官方数据
从 [holiday-cn](https://github.com/NateScarlet/holiday-cn) 项目下载对应年份的JSON文件

#### 2. 添加数据文件
将下载的文件（如 `2027.json`）放置到：
```
src/data/holidays/2027.json
```

#### 3. 更新索引文件
在 `src/data/holidays/index.js` 中添加：
```javascript
export { default as holidays2027 } from './2027.json'
```

#### 4. 更新数据映射
在 `src/utils/holidayData.js` 中：

**添加导入**:
```javascript
import holidays2027 from '../data/holidays/2027.json'
```

**更新映射对象**:
```javascript
const holidayDataMap = {
  // ... 现有年份
  2027: holidays2027.days || [],
}
```

**更新支持年份数组**:
```javascript
const supportedYears = [2007, 2008, ..., 2026, 2027]
```

#### 5. 验证数据
重启开发服务器，检查新年份是否在年份选择器中出现并正常工作。

### 数据格式说明
假期数据文件格式（参考 `holiday-cn` 项目）：
```json
{
  "year": 2027,
  "papers": ["..."],
  "days": [
    {
      "name": "元旦",
      "date": "2027-01-01",
      "isOffDay": true
    }
  ]
}
```
